## 🚀 Getting Started

```bash
git clone https://github.com/chauhanarjit07/Basics_of_DSA.git
cd Basics_of_DSA
```

Compile:

```bash
g++ filename.cpp -o output
```

Run:

**Windows**
```bash
output.exe
```

**Linux/macOS**
```bash
./output
```
