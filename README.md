# Basics in DSA

This repository contains various Data Structures and Algorithms (DSA) practice problems and solutions.

* The **`patterns`** folder includes a collection of programming pattern problems to help strengthen logic-building and problem-solving skills.
* The **`basics_maths`** folder contains solutions to fundamental mathematics problems, such as checking whether a number is prime, counting the number of digits in a number, reversing a number, finding GCD, and more.

This repository is intended for practice, learning, and improving DSA fundamentals. Contributions, suggestions, and improvements are always welcome!
